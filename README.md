# Focus Timer for Bangle.js 2

Focus Timer is a small Bangle.js 2 app for helping a child stay focused during a work session. The child starts a timer, and the watch periodically vibrates and asks how things are going: `TOP`, `MOYEN`, or `BLOQUE`.

## Features

- 10, 15, and 20 minute presets from the app start screen
- Filled circular countdown display
- Regular check-ins, default every 5 minutes
- Automatic screen wake and touch unlock when a check-in appears
- Color-coded responses and short help messages
- App Loader icon and launcher metadata
- Optional top-left `FT` widget that opens the app from clocks that show widgets
- Bangle.js settings screen for work duration and check interval

## Files

| File | Purpose |
|---|---|
| `focustimer/metadata.json` | App Loader metadata and storage manifest |
| `focustimer/app.js` | Main app, state machine, timer, UI rendering |
| `focustimer/widget.js` | Widget shown as `FT` in the widget bar |
| `focustimer/app-settings.js` | Settings menu |
| `focustimer/messages.json` | Editable feedback/help messages |
| `focustimer/app.png` | 48x48 App Loader icon preview |
| `focustimer/app-icon.js` | Espruino image string used as `focustimer.img` |
| `focustimer/tests/logic.test.js` | Node.js tests for pure helpers |

## Install

Use the Bangle.js App Loader with this folder as an app source, or install from files with the contents of `focustimer/`.

The launcher metadata is generated by the App Loader from `metadata.json`. The watch icon is uploaded from `app-icon.js` to `focustimer.img`.

## Development Checks

```bash
node focustimer/tests/logic.test.js
node --check focustimer/app.js
node --check focustimer/widget.js
node --check focustimer/app-icon.js
node -e "JSON.parse(require('fs').readFileSync('focustimer/metadata.json','utf8'))"
```
